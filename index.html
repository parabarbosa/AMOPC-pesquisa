Configuração da API do Google

Para usar este motor de busca, você precisa configurar uma API Key do Google Sheets.

Esta API Key será salva apenas localmente no seu navegador.

Como obter uma API Key do Google?
  1. Acesse o Google Cloud Console
  2. Crie um novo projeto ou selecione um existente
  3. Ative a API "Google Sheets API"
  4. Vá para "Credenciais" e crie uma "API Key"
  5. Cole a API Key no campo acima
